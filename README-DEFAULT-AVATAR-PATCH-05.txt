Patch 05 - Default Worker Avatar
تطبيق: صنايعي مطروح

الهدف:
إضافة صورة رمزية افتراضية احترافية تظهر تلقائيًا إذا لم يرفع الصنايعي صورة شخصية، أو إذا كان رابط الصورة مكسورًا.

الملفات المعدلة/المضافة:
- index.html
- worker.html
- admin.html
- service-worker.js
- icons/default-worker-avatar.png

ما تم تغييره:
1) الصفحة الرئيسية:
   - getWorkerImage() أصبحت ترجع /icons/default-worker-avatar.png عند عدم وجود صورة.
   - تم إضافة onerror على صورة كارت الصنايعي للرجوع للصورة الافتراضية لو الصورة الأصلية لم تُحمّل.

2) صفحة تفاصيل الصنايعي:
   - imgPath() أصبحت ترجع الصورة الافتراضية عند عدم وجود صورة.
   - تم إضافة fallback عند فشل تحميل صورة الصنايعي.

3) لوحة الإدارة:
   - wimg() أصبحت ترجع الصورة الافتراضية عند عدم وجود صورة.
   - تم إضافة fallback عند فشل تحميل صورة الصنايعي داخل كارت الإدارة.

4) PWA:
   - تم إضافة /icons/default-worker-avatar.png إلى قائمة الكاش.
   - تم رفع CACHE_NAME إلى sanay3i-matrouh-v4 لإجبار المتصفح على تحديث الكاش.

طريقة التركيب:
انسخ الملفات التالية إلى فولدر backend:

index.html              -> بدل القديم
worker.html             -> بدل القديم
admin.html              -> بدل القديم
service-worker.js       -> بدل القديم
icons/default-worker-avatar.png -> داخل فولدر icons

إذا لم يكن فولدر icons موجودًا، أنشئه أولًا.

التجربة:
1) افتح لوحة الإدارة.
2) افتح أي صنايعي بدون صورة أو امسح قيمة image من قاعدة البيانات للتجربة.
3) افتح الصفحة الرئيسية وصفحة تفاصيل الصنايعي.
4) يجب أن تظهر الصورة الرمزية الافتراضية بدل الصورة الفارغة.

ملاحظة:
بعد استبدال service-worker.js، قد تحتاج تعمل Hard Refresh أو Clear Site Data من المتصفح لو الصورة القديمة ظلت ظاهرة بسبب الكاش.
