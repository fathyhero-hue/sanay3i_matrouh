# Patch 13 — شارة موثّق للصنايعي

هذا الباتش يضيف ظهور حالة التحقق للعملاء بدون كشف أي صور بطاقة أو مستندات.

## الملفات المعدلة

انسخ الملفات التالية مكان الملفات القديمة:

- `index.html`
- `worker.html`
- `api/server.js`
- `service-worker.js`

وتأكد أن الصورة الاحتياطية موجودة كما هي:

- `icons/default-worker-avatar.png`

## ما الذي تغير؟

### 1. الصفحة الرئيسية

- يظهر Badge على كارت الصنايعي الموثق: `موثّق`.
- يظهر سطر ثقة داخل الكارت: `موثّق من الإدارة`.
- الصنايعي الموثق يظهر قبل غير الموثق في نتائج البحث.
- إضافة فلتر جديد: `موثّقين فقط`.

### 2. صفحة تفاصيل الصنايعي

- تظهر شارة واضحة: `موثّق من الإدارة`.
- لا يتم عرض أي بيانات بطاقة شخصية أو مسارات صور البطاقة للعميل.

### 3. API

- تم إضافة الحقل العام الآمن فقط:

```txt
identity_verified
```

إلى استعلامات العملاء العامة.

### 4. PWA Cache

تم رفع نسخة الكاش إلى:

```txt
sanay3i-matrouh-v12
```

## ملاحظات مهمة

- هذا الباتش يعتمد على Patch 12 لأن حالة التحقق تأتي من الحقل `identity_verified`.
- لا تحتاج لتشغيل SQL جديد إذا كنت شغّلت SQL الخاص بـ Patch 12 بالفعل.
- صور البطاقة الشخصية تظل خاصة للإدارة فقط ولا تظهر للعملاء.
