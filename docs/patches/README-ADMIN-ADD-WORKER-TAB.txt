ADMIN ADD WORKER TAB PATCH
==========================

تمت إضافة الآتي:

1) تاب داخل لوحة الإدارة:
   /admin -> إضافة صنايعي

2) صفحة إضافة صنايعي من الإدارة:
   /admin/add-worker

3) API خاص بالإضافة من الإدارة:
   POST /api/admin/workers/create

المميزات:
- نفس بيانات التسجيل الأساسية.
- رفع صورة شخصية.
- رفع صور أعمال.
- رفع وجه وظهر البطاقة اختياري من الإدارة.
- الموافقة والتفعيل مباشرة افتراضيًا.
- التوثيق لا يتفعل إلا إذا تم رفع وجه وظهر البطاقة واختيار "توثيق من الإدارة".
- منع تكرار رقم الهاتف أو الواتساب.
- إنشاء رقم طلب تلقائي SN-YYYY-00000.
- إضافة سجل في Admin Activity Log.
