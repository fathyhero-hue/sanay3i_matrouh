ADMIN_WORKER_ACTIONS_REBUILD_V6_PATCH

الغرض:
- حذف أزرار كارت الصنايعي القديمة المتعطلة عمليًا.
- إضافة زر واحد واضح داخل كل كارت: فتح لوحة الإجراءات.
- فتح Modal احترافي منظم يحتوي على: تعديل البيانات، مراجعة/تحقق، إدارة صور الأعمال، اشتراك وتجديد، رسائل واتساب، تفعيل/إيقاف، تمييز، حذف.
- إضافة event delegation احتياطي حتى تعمل الأزرار بعد إعادة رندر الكروت.

الملفات:
- admin.html
- service-worker.js

طريقة التركيب:
1) استبدل admin.html و service-worker.js داخل backend.
2) أوقف السيرفر ثم شغله: npm run dev
3) افتح http://localhost:3000/admin
4) نفذ Ctrl + Shift + R
5) لو بقيت ترى الأزرار القديمة فقط، افتح DevTools > Application > Service Workers > Unregister ثم Reload.

علامة نجاح التركيب:
يظهر داخل كارت الصنايعي نص: تم إعادة بناء الأزرار V6
ويظهر زر: فتح لوحة الإجراءات
