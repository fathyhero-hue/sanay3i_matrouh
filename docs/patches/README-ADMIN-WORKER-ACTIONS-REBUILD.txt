# ADMIN WORKER ACTIONS REBUILD PATCH

الغرض: حذف أزرار كارت الصنايعي القديمة داخل لوحة الإدارة وإعادة بنائها بشكل منظم واحترافي.

## ما تم تغييره
- إعادة بناء أزرار كارت الصنايعي داخل لوحة الإدارة في مجموعات واضحة:
  - إدارة أساسية
  - مراجعة وملفات
  - اشتراك ورسائل
  - إجراء حساس
- إصلاح أزرار:
  - تعديل البيانات
  - مراجعة التحقق
  - إدارة صور الأعمال
  - اشتراك وتجديد
  - رسائل واتساب
- صور الأعمال لا تُحمّل تلقائيًا، يتم تحميلها فقط عند الضغط على زر إدارة صور الأعمال لتسريع اللوحة.
- إضافة نظام موحد لتحديد الصنايعي من id / رقم الطلب / الهاتف / واتساب.

## التركيب
استبدل ملف admin.html فقط داخل مجلد backend.

لا يحتاج هذا الباتش SQL ولا تعديل api/server.js.

## بعد التركيب
1. Ctrl + C
2. npm run dev
3. افتح http://localhost:3000/admin
4. Ctrl + F5
5. جرّب أزرار كارت الصنايعي.
