Patch 23: تحسين سرعة وأداء الصفحة الرئيسية

الملفات التي يتم استبدالها:
- index.html
- service-worker.js

لا يوجد SQL جديد.
لا يوجد تعديل مطلوب في api/server.js.

ما تم تحسينه:
1. إضافة Skeleton Loading لكروت الصنايعية أثناء تحميل البيانات.
2. إضافة lazy loading و async decoding لصور الصنايعية في الصفحة الرئيسية.
3. عرض كروت الصنايعية على دفعات بدلاً من رسم كل النتائج مرة واحدة.
4. استخدام DocumentFragment لتقليل إعادة الرسم داخل المتصفح.
5. إضافة Debounce للبحث حتى لا يتم الفلترة مع كل ضغطة حرف فورًا.
6. تقليل ضغط طلبات التقييمات بجلبها على دفعات صغيرة بدل تنفيذ كل الطلبات مرة واحدة.
7. تحسين تجربة الموبايل أثناء التحميل.
8. تحديث كاش PWA إلى sanay3i-matrouh-v25.

طريقة التركيب:
1. انسخ index.html مكان الملف القديم في جذر المشروع.
2. انسخ service-worker.js مكان الملف القديم في جذر المشروع.
3. شغل السيرفر:
   npm start
4. افتح الصفحة الرئيسية واعمل Ctrl + F5.

اختبارات مقترحة:
- افتح الصفحة الرئيسية على الموبايل.
- جرّب البحث بسرعة.
- جرّب فلتر الحرفة والمنطقة.
- جرّب رابط /trade/سباك.
- جرّب رابط /area/علم%20الروم.
- تأكد أن الكروت تظهر تدريجيًا وأن الصور تحمل بدون بطء واضح.
