صنايعي مطروح - WORKER PAGE UX PATCH
====================================

الغرض:
تحسين صفحة الصنايعي بدون تغيير قاعدة البيانات وبدون SQL.

الملفات:
- worker.html فقط

الإضافات:
1) شريط ثقة سريع أعلى صفحة الصنايعي:
   - حالة التواصل
   - حالة التوثيق
   - تاريخ الانضمام

2) تحسين أزرار التواصل:
   - اتصال
   - واتساب
   - نسخ الرقم
   - مشاركة الصفحة
   - بلاغ عن البيانات

3) تحسين الموبايل:
   - شريط ثابت أسفل الشاشة بأربعة أزرار صغيرة وواضحة
   - تنظيم أفضل في الوضع الطولي

4) إضافة صندوق نصي للتنبيه:
   - الاتفاق على السعر والتفاصيل قبل بداية الشغل
   - استخدام البلاغ لو البيانات غير صحيحة

5) إضافة قسم صنايعية مشابهين:
   - يظهر صنايعية من نفس الحرفة أو المنطقة
   - يعتمد على /api/workers الموجود بالفعل
   - لا يحتاج API جديد

طريقة التركيب:
1. استبدل worker.html في مجلد المشروع:
   C:\Users\HERO\Desktop\sanay3i_matrouh\backend\worker.html

2. شغل محليًا:
   npm run dev

3. افتح صفحة صنايعي:
   http://localhost:3000/worker/14

4. اعمل Ctrl + F5

5. اختبر:
   - زر الاتصال
   - زر واتساب
   - زر نسخ الرقم
   - زر مشاركة
   - زر البلاغ
   - قسم صنايعية مشابهين
