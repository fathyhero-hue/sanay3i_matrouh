# Patch 19 — صفحة نجاح بعد التسجيل + رقم طلب للصنايعي

## الملفات
انسخ الملفات التالية مكان القديمة:

- register.html
- admin.html
- api/server.js
- service-worker.js

## SQL مهم
شغّل الملف التالي مرة واحدة داخل Supabase SQL Editor:

SUPABASE-REGISTRATION-CODE-PATCH-19.sql

سيضيف العمود:

registration_code

ويملأ الأكواد للصنايعية القدامى بصيغة مثل:

SN-2026-00025

## الاختبار
1. شغّل السيرفر:

npm start

2. افتح صفحة التسجيل:

http://localhost:3000/register

3. سجّل صنايعي جديد ببيانات ورقم غير مكرر.
4. بعد الإرسال يجب أن تظهر رسالة نجاح وبداخلها رقم الطلب.
5. افتح لوحة الإدارة وابحث برقم الطلب.

## ملاحظات
- لا يتم عرض رقم الطلب في صفحة العميل العامة.
- يظهر رقم الطلب داخل كارت الصنايعي في لوحة الإدارة فقط.
- إذا نسيت تشغيل SQL سيظل السيرفر يولد رقم الطلب في الرد، لكن لن يتم حفظه داخل قاعدة البيانات حتى يتم تشغيل SQL.
