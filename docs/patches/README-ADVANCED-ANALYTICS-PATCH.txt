# ADVANCED ANALYTICS PATCH - صنايعي مطروح

يضيف هذا الباتش تحليلات احترافية داخل لوحة الإدارة.

## الملفات
- api/server.js
- admin.html
- index.html

## الإضافات
- إجمالي الأحداث ونسبة تحويل المشاهدة إلى تواصل.
- أفضل الصنايعية حسب الاتصال والواتساب.
- أكثر الحرف طلبًا.
- أكثر المناطق طلبًا.
- اختيارات الزوار للحرف والمناطق من الفلاتر.
- أكثر الصفحات زيارة.
- ملخص يومي لآخر 31 يوم كحد أقصى.
- فجوات التغطية: حرف غير موجودة في مناطق معينة.

## هل يحتاج SQL؟
لا، بشرط أن جدول analytics_events موجود بالفعل من المرحلة السابقة.

## طريقة التركيب
1. استبدل الملفات الثلاثة داخل backend.
2. شغّل محليًا: npm run dev
3. افتح /admin ثم تبويب التحليلات.
4. جرّب من الصفحة الرئيسية اختيار حرفة/منطقة ثم افتح التحليلات مرة أخرى.

## ملاحظة
أرقام اختيارات الحرف والمناطق ستبدأ من وقت تركيب هذا الباتش فقط، أما ضغطات الاتصال والواتساب القديمة ستظل تظهر كما هي.
