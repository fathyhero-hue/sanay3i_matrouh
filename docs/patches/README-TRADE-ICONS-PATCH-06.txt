Patch 06 - أيقونات الحرف في الصفحة الرئيسية

الهدف:
إضافة قسم جديد في الصفحة الرئيسية يعرض كل الحرف على شكل أيقونات احترافية. عند الضغط على أي حرفة يتم فلترة الصنايعية وعرض الحرفيين الموجودين في هذه الحرفة مباشرة.

الملفات المعدلة:
- index.html
- service-worker.js

طريقة التركيب:
1) انسخ index.html مكان الملف القديم.
2) انسخ service-worker.js مكان الملف القديم.
3) شغل المشروع محليًا:
   npm start
4) افتح:
   http://localhost:3000

ما الذي تغير؟
- إضافة قسم "اختيار سريع حسب الحرفة" بعد الإحصائيات وقبل دليل الصنايعية.
- كل حرفة تظهر بكارت وأيقونة.
- الكارت يعرض عدد الصنايعية المتاحين داخل الحرفة.
- الضغط على أي حرفة يمسح البحث والمنطقة، يختار الحرفة، ويذهب تلقائيًا لقائمة الصنايعية.
- أيقونة كل حرفة يتم اختيارها تلقائيًا حسب اسم الحرفة، مثل سباك، كهربائي، نجار، نقاش، فني تكييف، حداد، سيراميك، كاميرات، دش، إنترنت، أجهزة، تنظيف، نقل عفش...
- رفع نسخة كاش PWA إلى sanay3i-matrouh-v5 حتى يظهر التحديث بعد النشر.

ملاحظة:
لو ظهرت أيقونة عامة لبعض الحرف، هذا طبيعي إذا كان اسم الحرفة غير موجود ضمن قاموس الأيقونات. يمكن لاحقًا إضافة خريطة أيقونات أدق حسب أسماء الحرف الموجودة عندك في Supabase.
