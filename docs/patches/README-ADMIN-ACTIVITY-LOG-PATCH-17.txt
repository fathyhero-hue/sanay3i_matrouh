Patch 17 - سجل نشاط الإدارة
=========================

الهدف:
إضافة تبويب جديد في لوحة الإدارة باسم "سجل النشاط" لمتابعة آخر العمليات الإدارية.

الملفات المطلوب نسخها مكان القديمة:
1) admin.html
2) api/server.js
3) service-worker.js

ملف SQL المطلوب تشغيله مرة واحدة:
SUPABASE-ADMIN-ACTIVITY-LOG-PATCH-17.sql

طريقة تشغيل SQL:
Supabase > SQL Editor > New Query > Paste > Run

العمليات التي يتم تسجيلها:
- تعديل بيانات صنايعي
- اعتماد / إلغاء اعتماد صنايعي
- تفعيل / إيقاف صنايعي
- تمييز / إلغاء تمييز صنايعي
- مراجعة التحقق
- تجديد الاشتراك
- إضافة / حذف صور أعمال
- اعتماد / إلغاء اعتماد / حذف تقييم
- إضافة / حذف حرفة
- إضافة / حذف منطقة
- حذف صنايعي

ملاحظات:
- لو لم يتم تشغيل SQL، لن يتعطل الموقع، لكن تبويب سجل النشاط سيعرض تنبيهًا بأن الجدول غير موجود.
- تم تحديث كاش PWA إلى: sanay3i-matrouh-v16
- لا يظهر سجل النشاط للعملاء، فقط داخل لوحة الإدارة.
