Patch 15 - تحسين ترتيب وظهور الصنايعية في الصفحة الرئيسية

الملفات المعدلة:
- index.html
- service-worker.js

الملفات المرفقة بدون تعديل وظيفي:
- api/server.js
- vercel.json
- icons/default-worker-avatar.png

ما الذي تم إضافته:
1) ترتيب افتراضي أذكى للصنايعية:
   - المميز Featured أولاً
   - الموثق من الإدارة ثانياً
   - الاشتراك النشط
   - الأعلى تقييماً
   - الأحدث تسجيلًا

2) خيارات ترتيب في الصفحة الرئيسية:
   - الأفضل أولًا
   - المميزين أولًا
   - الموثقين أولًا
   - الأعلى تقييمًا
   - الأحدث

3) فلتر جديد:
   - المميزين فقط

4) الفلتر القديم مستمر:
   - موثقين فقط

5) لا يوجد SQL جديد مطلوب.

طريقة التركيب:
انسخ الملفات التالية مكان القديمة:
- index.html
- service-worker.js

اختياريًا، يمكنك نسخ api/server.js و vercel.json إذا أردت توحيد النسخة، لكن لا يوجد تعديل مطلوب في السيرفر لهذا الباتش.

بعد النسخ:
1) أوقف السيرفر Ctrl + C
2) شغله مرة أخرى:
   npm start
3) افتح الصفحة الرئيسية:
   http://localhost:3000
4) اضغط Ctrl + F5 لتفريغ كاش المتصفح.
