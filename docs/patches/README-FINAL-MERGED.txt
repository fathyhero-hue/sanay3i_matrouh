# صنايعي مطروح - نسخة مدمجة نهائية

هذه نسخة قديمة تم دمج كل التعديلات الجديدة عليها وتنظيفها للنشر.

## أهم ما تم دمجه
- صفحة حالة الطلب `/status` ورقم الطلب `SN-YYYY-00000`.
- تسجيل صنايعي بخطوات مع إلزام رفع وجه وظهر البطاقة.
- مراجعة التحقق من الإدارة مع إمكانية اعتماد الإدارة حتى بدون بطاقة للطلبات القديمة/الإضافة اليدوية.
- صفحة إضافة صنايعي من الإدارة `/admin/add-worker`.
- تسريع الصفحة الرئيسية بإحصاء سريع وعدّاد صنايعية.
- تلخيص التقييمات من API واحد.
- اشتراكات وتجديد الاشتراك وسجل المدفوعات.
- سجل نشاط الإدارة.
- شارة موثّق وشارة مميز.
- صور الأعمال والتقييمات.
- fallback avatar.
- تحسينات الموبايل الطولي Patch 25.
- إعداد Vercel يفصل الصفحات Static عن API.

## التشغيل المحلي
```bash
npm install
npm run dev
```
ثم افتح:
```txt
http://localhost:3000
http://localhost:3000/admin
http://localhost:3000/admin/add-worker
http://localhost:3000/worker/24
```

## النشر على Vercel
- Root Directory: اتركه فاضيًا أو `./` إذا كان هذا المجلد هو جذر الريبو.
- لا تكتب `backend` إلا إذا كان GitHub يحتوي فولدر داخله باسم backend.
- أضف متغيرات البيئة في Vercel: `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`, `ADMIN_PASSWORD`, `ADMIN_SESSION_SECRET`, `ADMIN_SESSION_DAYS`, `SUPABASE_BUCKET`, `SUPABASE_ID_BUCKET`.

## ملاحظات أمنية
- لا ترفع `.env` إلى GitHub.
- صور البطاقة محفوظة في bucket خاص `identity-docs` ولا تظهر للعملاء.
- ملفات `.git` و `node_modules` غير مرفقة في هذه النسخة لتقليل الحجم ومنع مشاكل النشر.
