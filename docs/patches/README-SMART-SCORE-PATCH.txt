SMART SCORE PATCH - صنايعي مطروح

الهدف:
- ترتيب الصنايعية بذكاء بدل الترتيب العادي.
- بدون إنشاء جداول جديدة وبدون تشغيل SQL.

الملفات:
- api/server.js
- index.html

ما الذي يفعله؟
1) يضيف smart_score لكل صنايعي ظاهر في /api/workers.
2) يعتمد الترتيب على:
   - الصنايعي المميز
   - الصنايعي الموثق
   - الاشتراك النشط
   - متوسط التقييم وعدد التقييمات
   - ضغطات الاتصال والواتساب خلال آخر 30 يومًا
   - زيارات صفحة الصنايعي خلال آخر 30 يومًا
   - الأحدث
3) يضيف شارة صغيرة في كارت الصنايعي توضح الترتيب الذكي.
4) يضيف endpoint إداري للمراجعة:
   /api/admin/smart-score

طريقة التركيب:
1) استبدل:
   api/server.js
   index.html
2) شغل محليًا:
   npm run dev
3) افتح:
   http://localhost:3000
4) تأكد أن ترتيب الصنايعية أصبح حسب الأفضل تلقائيًا.
5) يمكن مراجعة النتيجة من:
   http://localhost:3000/api/admin/smart-score
   بعد تسجيل دخول الإدارة.

ملاحظة:
- لا يحتاج SQL.
- يستخدم analytics_events إذا كان موجودًا، ولو لم تكن هناك أحداث كافية سيظل الترتيب يعمل بالتمييز/التوثيق/التقييم/الأحدث.
- يمكن تغيير نافذة التحليلات من Vercel env:
  SMART_SCORE_ANALYTICS_DAYS=30
