# WhatsApp Card Button Fix V3

هذا الباتش يصلح زر "رسائل واتساب" الموجود أسفل كارت الصنايعي داخل لوحة التحكم بعد باتش تسريع الإدارة.

## سبب المشكلة
بعد تحسين الأداء، بعض الكروت قد ترسل رقم طلب أو رقم غير مطابق للـ id الحقيقي في allWorkers، لذلك نافذة واتساب لا تجد بيانات الصنايعي.

## الإصلاح
- البحث عن الصنايعي بأكثر من مفتاح:
  - id الحقيقي
  - رقم الطلب مثل SN-2026-00033
  - آخر رقم في رقم الطلب مثل 33
  - رقم الهاتف
  - رقم الواتساب
  - بيانات الكارت نفسه
- تعديل زر واتساب داخل الكارت لاستخدام openWhatsAppModalFromButton.
- الحفاظ على نفس نظام الإرسال الفردي والجماعي وسجل الرسائل بدون تغيير API أو SQL.

## التركيب

اضغط زر "رسائل واتساب" أسفل أي كارت صنايعي.
أو من Console جرّب:
testOpenWorkerWhatsappModal("33")
