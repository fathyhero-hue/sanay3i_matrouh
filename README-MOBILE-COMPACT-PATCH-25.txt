# Patch 25 - Compact Mobile Portrait UI

هذا الباتش يركز على جعل التطبيق أكثر ملاءمة لشاشات الموبايل الطولية، مع تصغير الأيقونات والكتابات وتنظيم المسافات.

## الملفات المعدلة

- index.html
- worker.html
- register.html
- status.html
- admin.html
- service-worker.js

## أهم التحسينات

1. تصغير الهيدر واللوجو ومنيو الموبايل.
2. تصغير كروت الحرف والمناطق وتحويلها إلى 3 أعمدة على الموبايل بدل كروت كبيرة.
3. جعل كروت الصنايعية في الرئيسية بشكل قائمة مضغوطة مناسبة للتمرير الطولي.
4. تقليل حجم النصوص والمسافات داخل الفلاتر والبحث.
5. ترتيب صفحة التسجيل بنظام خطوات بشكل أخف على الموبايل.
6. تحسين صفحة حالة الطلب لتكون مختصرة ومنظمة.
7. تحسين صفحة تفاصيل الصنايعي على الموبايل.
8. تحسين لوحة الإدارة على الموبايل بتبويبات قابلة للسحب وأزرار أصغر.

## طريقة التركيب

انسخ الملفات التالية مكان القديمة:

```txt
index.html
worker.html
register.html
status.html
admin.html
service-worker.js
```

ثم شغل السيرفر من جديد:

```bash
npm start
```

وافتح التطبيق على الموبايل أو من Chrome DevTools وضع Mobile Portrait ثم اعمل:

```txt
Ctrl + F5
```
