نسخة نهائية مستقرة - صنايعي مطروح Local Version
تاريخ التجميع: 2026-06-04 12:01:55

الملفات داخل النسخة:
- server.js
- index.html
- register.html
- worker.html
- admin.html

مكان النسخ:
انسخ الملفات الخمسة داخل فولدر:
Desktop\sanay3i_matrouh\backend

مهم:
لا تحذف الملفات التالية من فولدر backend:
- sanaiey.db
- uploads folder
- node_modules
- package.json
- package-lock.json
- style.css لو موجود ومستخدم عندك

تشغيل السيرفر:
cd %USERPROFILE%\Desktop\sanay3i_matrouh\backend
node server.js

روابط التشغيل:
http://localhost:3000
http://localhost:3000/register
http://localhost:3000/admin
http://localhost:3000/worker/1

كلمة سر الإدارة الحالية:
123456

المميزات الموجودة في هذه النسخة:
1) صفحة رئيسية احترافية.
2) بحث نصي.
3) بحث سريع بالحرف المشهورة.
4) فلتر حسب الحرفة.
5) فلتر حسب المنطقة.
6) ترتيب حسب:
   - الافتراضي
   - الأعلى تقييمًا
   - الأحدث
   - المميزين أولًا
7) كارت صنايعي فيه:
   - صورة
   - اسم
   - حرفة
   - منطقة
   - تقييم
   - اتصال
   - واتساب
   - مشاركة
8) تسجيل صنايعي:
   - رقم اتصال
   - رقم واتساب
   - اختيار الحرفة والمنطقة من السيرفر
   - قص الصورة الأساسية
   - قص صور الأعمال
   - حد أقصى 5 صور أعمال
9) أول شهر مجاني تلقائيًا.
10) الصنايعي لا يظهر إلا بعد موافقة الإدارة.
11) صفحة تفاصيل محسنة:
   - بيانات منظمة
   - تقييم واضح
   - صور أعمال
   - ريفيوهات
   - اتصال وواتساب
   - مشاركة
12) لوحة إدارة:
   - موافقة / إلغاء موافقة
   - تفعيل / إيقاف
   - تمييز
   - تعديل
   - حذف
   - اشتراكات شهر / نصف سنة / سنة
   - إدارة الحرف
   - إدارة المناطق
   - إدارة التقييمات
   - حذف صور الأعمال
   - إشعارات الإدارة
   - نسخ احتياطي
   - تصدير Excel/CSV

لو حصلت مشكلة بعد نسخ الملفات:
1) اقفل السيرفر Ctrl + C
2) شغل:
   node server.js
3) افتح الصفحة واعمل:
   Ctrl + F5

ملاحظات:
- هذه نسخة Local للتجربة على الكمبيوتر.
- عند الرفع الحقيقي لاحقًا هنحتاج نراجع حماية لوحة الإدارة والاستضافة وقاعدة البيانات.
