# Patch 18 — تحويل صفحة التسجيل إلى خطوات

## الملفات المطلوبة
انسخ الملفات التالية مكان القديمة:

- register.html
- service-worker.js

## ما الجديد؟

تم تحويل تسجيل الصنايعي إلى 4 خطوات واضحة:

1. البيانات الأساسية: الاسم ورقم الاتصال والواتساب.
2. الحرفة والمنطقة والوصف.
3. رفع البطاقة الشخصية وجه وظهر.
4. الصورة الأساسية وصور الأعمال ثم إرسال الطلب.

## ملاحظات

- شرط رفع البطاقة ما زال إجباريًا في التسجيل الجديد.
- منع تكرار رقم الهاتف أو الواتساب ما زال موجودًا.
- لا يوجد SQL جديد مطلوب.
- تم تحديث كاش PWA إلى sanay3i-matrouh-v17.

## بعد التركيب

شغل السيرفر:

npm start

ثم افتح:

http://localhost:3000/register

واعمل Ctrl + F5.
