# صنايعي مطروح

تطبيق ويب/PWA لإدارة وعرض الصنايعية داخل محافظة مطروح، مع لوحة تحكم للإدارة ونظام تسجيل ومراجعة الصنايعية وخدمات التواصل.

## أهم المميزات

* عرض الصنايعية حسب الحرفة والمنطقة.
* صفحة تفاصيل لكل صنايعي.
* تسجيل صنايعي جديد.
* مراجعة واعتماد الصنايعية من لوحة الإدارة.
* إدارة بيانات الصنايعي والصور والاشتراكات.
* شات داخلي بين الصنايعي والإدارة.
* شات خدمة العملاء داخل التطبيق.
* إرسال رسائل واتساب من لوحة الإدارة.
* نظام بلاغات وتحليلات وسجل عمليات.

## التشغيل المحلي

```bash
npm install
npm start
```

أو:

```bash
node server.js
```

ثم افتح:

```txt
http://localhost:3000
```

## تنظيم التعديلات

* يتم استخدام Git branches لأي ميزة أو إصلاح جديد.
* يتم تسجيل أهم التغييرات في ملف `CHANGELOG.md`.
* ملفات توثيق الباتشات القديمة موجودة داخل:

```txt
docs/patches
```

## تنبيه أمني

لا يتم رفع أو مشاركة الملفات الحساسة مثل:

```txt
.env
signing-key-info.txt
keystore
tokens
passwords
```
