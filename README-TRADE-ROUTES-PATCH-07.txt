Patch 07 - روابط مباشرة لكل حرفة

الهدف:
- أيقونة الحرفة في الصفحة الرئيسية لا تكتفي بالفلترة فقط.
- عند الضغط عليها يتغير الرابط إلى /trade/اسم-الحرفة.
- يمكن نسخ الرابط ومشاركته مع أي شخص ليفتح مباشرة على صنايعية نفس الحرفة.

الملفات المعدلة:
- index.html
- api/server.js
- vercel.json
- service-worker.js

طريقة التركيب:
1) انسخ index.html مكان القديم.
2) انسخ api/server.js داخل فولدر api مكان القديم.
3) انسخ vercel.json مكان القديم.
4) انسخ service-worker.js مكان القديم.
4) شغل محليًا: npm start

اختبار سريع:
- افتح http://localhost:3000
- اضغط على أي حرفة مثل سباك.
- الرابط يتحول إلى /trade/%D8%B3%D8%A8%D8%A7%D9%83
- انسخ الرابط وافتحه في تبويب جديد.
- لازم يفتح مباشرة على صنايعية هذه الحرفة.
