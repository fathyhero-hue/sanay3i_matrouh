باتش 03 - Dashboard إحصائيات تشغيلية ومالية

الهدف:
إضافة تبويب جديد داخل لوحة الإدارة باسم "الإحصائيات" يعرض ملخصًا سريعًا لحالة التطبيق والاشتراكات والمدفوعات.

الملفات التي يجب نسخها:
1) admin.html
   انسخه بدل ملف admin.html الموجود في فولدر المشروع.

2) api/server.js
   انسخه بدل ملف api/server.js الموجود في فولدر api.

المميزات الجديدة:
- تبويب "الإحصائيات" داخل لوحة الإدارة.
- إجمالي الصنايعية.
- الموافق عليهم.
- بانتظار الموافقة.
- الصنايعية المميزين.
- الاشتراكات النشطة.
- الاشتراكات التي قاربت الانتهاء خلال 7 أيام.
- الاشتراكات المنتهية.
- التقييمات المنتظرة.
- إجمالي المدفوعات المسجلة.
- مدفوعات الشهر الحالي.
- عدد عمليات الدفع.
- متوسط عملية الدفع.
- أعلى الحرف عددًا.
- أعلى المناطق عددًا.
- آخر 10 عمليات تجديد ودفع.

API الجديد:
GET /api/admin/dashboard-stats

ملاحظات:
- هذا المسار محمي بنفس نظام تسجيل دخول الإدارة.
- يعتمد جزء المدفوعات على جدول subscription_payments الذي تم إنشاؤه في باتش الاشتراكات.
- إذا لم يكن جدول subscription_payments موجودًا، ستظهر رسالة داخل Dashboard بدل أن يتعطل التطبيق.

طريقة التشغيل المحلي:
npm start

ثم افتح:
http://localhost:3000/admin

وسجل الدخول ثم اضغط تبويب:
الإحصائيات

اختبارات تمت:
- فحص syntax لملف api/server.js.
- فحص syntax للـ JavaScript داخل admin.html.
