PERFORMANCE_SPEED_OPTIMIZATION_PATCH

الهدف:
- تسريع فتح الصفحة الرئيسية.
- تسريع فتح لوحة الإدارة.
- منع تحميل صور أعمال كل الصنايعية عند فتح الإدارة.
- منع تحميل التبويبات الثقيلة إلا عند فتحها.
- تقليل طلبات تقييمات الصنايعية من الصفحة الرئيسية.
- إضافة Cache آمن لبيانات الصفحة الرئيسية.
- تحديث Service Worker لإجبار المتصفح على نسخة جديدة.

الملفات:
- api/server.js
- admin.html
- index.html
- service-worker.js
- SUPABASE-PERFORMANCE-INDEXES-PATCH.sql

   - الصفحة الرئيسية
   - لوحة الإدارة
   - تبويب رسائل واتساب
   - تبويب البلاغات
   - صفحة صنايعي
