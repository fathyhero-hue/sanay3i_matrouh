# Patch 16 — منع تكرار تسجيل نفس الصنايعي

## الهدف
منع تسجيل أكثر من صنايعي بنفس رقم الهاتف أو رقم الواتساب، مع إظهار تنبيه داخل لوحة الإدارة لو فيه بيانات قديمة مكررة بالفعل.

## الملفات المطلوب نسخها
- register.html
- admin.html
- api/server.js
- service-worker.js

## ما الذي تغير؟

### 1) صفحة التسجيل
قبل إرسال الطلب، الصفحة تفحص الرقم مع السيرفر. لو رقم الاتصال أو الواتساب مسجل قبل كده، يظهر للصنايعي تنبيه ولا يتم إرسال الطلب.

### 2) السيرفر
أضيف فحص إجباري داخل `/api/register` و `/api/sanaieya` و `/api/workers`. حتى لو أحد حاول يتجاوز الواجهة، السيرفر سيرفض التسجيل المكرر.

### 3) لوحة الإدارة
لو فيه صنايعية قدامى مكررين بنفس الرقم، يظهر داخل كارت الصنايعي تنبيه واضح: "يوجد صنايعي آخر بنفس رقم الهاتف أو الواتساب".

### 4) API جديد
`GET /api/workers/check-duplicate?phone=...&whatsapp=...`

## طريقة التركيب
انسخ الملفات مكان القديمة، ثم أعد تشغيل السيرفر:

```bash
npm start
```

بعدها اعمل Hard Refresh:

```txt
Ctrl + F5
```

## SQL
لا يوجد SQL جديد مطلوب.

## ملاحظة
هذا الباتش لا يحذف الصنايعية المكررين القدامى. هو فقط يظهر تنبيه للإدارة ويمنع التكرار الجديد.
